# d3-function-plotter

Тестовое задание: React + TypeScript + D3.  
Два графика: sin(x) и tan(x). Есть оси, пан и зум, и сохранение текущего положения в URL, light/dark/system themes.

## Стек

- React
- TypeScript
- Vite
- D3.js

## Что сделано

- Отрисовка графика sin(x) и tan(x) в SVG
- Оси X/Y через d3.axis
- Drag через d3.zoom
- Зум по X / по Y / по двум осям
- Reset к дефолтному масштабу/позиции
- Сохранение x/y domain в query params (пример: ?x=0,20&y=-10,10)
- Для tan(x) есть фильтрация точек, чтобы не соединялись асимптоты

## Управление

- Drag мышкой по области графика: пан
- Zoom in / zoom out по отдельным осям + общий зум
- Reset: вернуть начальное состояние

## Как работает (коротко)

- React держит домены (xDomain/yDomain) в state
- D3 в useEffect пересчитывает scale/axis/path и обновляет элементы
- d3.zoom используется чтобы ловить drag и получать transform, дальше я перевожу это в сдвиг домена

## URL

Позиция и масштаб пишутся в URL, чтобы можно было:

- обновить страницу и сохранить вид
- отправить ссылку другому человеку
